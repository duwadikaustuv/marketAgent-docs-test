# Overview

import { Diagram } from "../components/Diagram";

MarketAgents is a distributed **agent orchestration framework** designed to simulate and manage market scenarios with AI-driven agents. Key features include:

- **Parallel LLM Inference**
- **Immutable Conversation State**
- **Integrated Tools**
- **Flexible Outputs**
